┌──────────────────────────────────────────────────────────────────────┐
│  ⚠️  ATENÇÃO: VÍDEO > 100MB?                                         │
└──────────────────────────────────────────────────────────────────────┘

    Ver tamanho:
      $ du -h public/video-boi.mp4
    
    Se > 100MB, GitHub NÃO aceita!
    
    SOLUÇÃO 1: Git LFS
    ─────────────────────────────────────────────────────────────────
      $ sudo apt install git-lfs
      $ git lfs install
      $ git lfs track "*.mp4"
      $ git add .gitattributes
      $ git add public/video-boi.mp4
      $ git commit -m "Vídeo com LFS"
      $ git push origin main
    
    SOLUÇÃO 2: Hospedar externamente
    ─────────────────────────────────────────────────────────────────
      • YouTube (não listado)
      • Google Drive
      • Cloudflare R2
      
      Depois alterar linha 336 do index.html:
        <video src="URL_EXTERNA">
