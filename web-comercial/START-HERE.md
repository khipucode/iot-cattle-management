## ⚠️ VÍDEO MAIOR QUE 100MB?

### **Ver tamanho:**
```bash
du -h public/video-boi.mp4
```

### **Se for > 100MB:**

**Opção 1: Git LFS**
```bash
sudo apt install git-lfs
git lfs install
git lfs track "*.mp4"
git add .gitattributes
git add public/video-boi.mp4
git commit -m "Vídeo com LFS"
git push origin main
```

**Opção 2: Hospedar externamente**
- YouTube (não listado)
- Google Drive
- Editar linha 336 do `index.html` com URL externa
