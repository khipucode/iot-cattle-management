## ✨ Sobre o Projeto

**iTrace** é um site institucional moderno para apresentação de um sistema de rastreamento de bovinos via RFID, com tecnologia de IA e Visão Computacional utilizando drones para monitoramento aéreo inteligente do gado.

### 🎯 Tecnologias Utilizadas

- **HTML5** puro
- **CSS3** puro (sem frameworks!)
- **JavaScript** vanilla (sem bibliotecas pesadas!)
- **Lucide Icons** (via CDN)

### 🚀 Por que HTML/CSS/JS puro?

- ⚡ **10x mais rápido** que React
- 📦 **10x menor** (~50KB vs ~500KB)
- 🎯 **SEO perfeito** (HTML direto)
- 🔧 **Zero build** necessário
- 🌐 **Hospeda em qualquer lugar**
- 💚 **Mesma beleza visual!**

## 🖥️ Como Usar

### **Opção 1: Abrir Direto no Navegador**

```bash
# Simplesmente abra o arquivo index.html
firefox index.html
# OU
google-chrome index.html
```

### **Opção 2: Servidor Local (Python)**

```bash
python3 -m http.server 8000
# Acesse: http://localhost:8000
```

### **Opção 3: Servidor Local (Node)**

```bash
npx http-server
```
